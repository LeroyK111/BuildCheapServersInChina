# 自建机房方案

本方案，仅适合小公司，家庭，创业团队等。免除租用云服务or拉取宽带专线等高昂成本。

与之相对的，防火墙和软件安全都必须自己负责。

**仅适合有能力折腾的人**

申请成功后，需要**设置防火墙，开放端口等**。



3.自建中转服务器（也需要一台带有公网ip的服务器）
